# Project Name

2D Platformer made with Godot
25 March 2023

Description

- A and D to move left and right

- W to jump, hold it for a higher jump

- Space to shoot a fireball

## Implementation

- most sprites made by me

- 3 levels

- music and sound effects

- light 2d used in level 3
